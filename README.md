
## Description

Додаток, що розробляється - фінансовий менеджер, що допомагає вести облік фінансів
власника.

```bash

# run database in docker
$ npm run start:docker

# run application
$ npm run start

# watch mode
$ npm run start:dev

```

Link for Swagger documentation [Swagger](http://localhost:3000/api#/).
